# EventManager

A mobile app for managing event production operations — built for an Israeli event production company, with a multi-tenant SaaS version planned as the next phase.

## What it does

EventManager handles the day-to-day operational needs of running events: role-based access for different staff types, worker-facing views for on-the-ground teams, and payment handling, all in a single mobile app.

**Core features:**
- Authentication and role-based routing (different views/permissions for admins vs. workers)
- Payment processing
- Worker-facing views for event-day operations
- Built with a future multi-tenant SaaS structure in mind, so multiple event production companies could eventually run on the same platform

## Stack

- **Frontend:** React Native + Expo
- **Backend:** Supabase (auth, database)
